# Post_Covid_Data
An end-to-end data project involving Excel-based data cleaning, sorting, and organization, with Power BI dashboards used to visualize key metrics, identify trends, and communicate insights effectively.

## Tools Used
- Microsoft Excel
- Microsoft Power BI
- Tables
- Filters
- Sorting techniques
- Graphs and visual elements

## What This Project Shows
- Cleaning and preparing raw data for analysis
- Sorting and managing large datasets efficiently
- Organizing structured data for meaningful analysis
- Creating clear visualizations to highlight trends and insights
